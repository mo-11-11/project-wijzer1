DevSkills - GIT

In deze module leer je alles over versiebeheer en hoe je het kunt gebruiken om beter te werken aan softwareprojecten. Git helpt je om jouw code op orde te houden en makkelijk samen te werken met anderen.

Wat ga je leren?

Tijdens deze cursus leer je stap voor stap hoe je Git kunt gebruiken. Je leert wat een repository is, hoe je bestanden toevoegt, commits maakt en hoe je jouw werk online zet met push. Ook leer je hoe branches werken en hoe je met anderen kunt samenwerken zonder dat de code in de war raakt. Je leert de basis van veelgebruikte commando's, zoals git add , git commit en git push. Daarnaast leer je hoe je conflicten oplost en wat de beste manier is om een project te organiseren. In de lessen leer je ook over workflows zoals Gitflow en hoe Git samenwerkt met tools zoals GitHub of Bitbucket. Na deze module kun je jouw werk professioneel beheren en beter samenwerken met je team.

Leeruitkomsten

  • Je weet wat versiebeheer is en waarom het belangrijk is.
  • Je kunt met Git repositories aanmaken en beheren.
  • Je kent de belangrijkste Git-commando's en hun functie.
  • Je kunt samenwerken via branches en merges.
