Database - SQL beheren

In deze module leer je hoe je gegevens goed kan organiseren, beheren en beveiligen in een applicatie. Je leert stap voor stap wat een database eigenlijk doet en waarom het belangrijk is om data veilig en netjes op te slaan.

Wat ga je leren?

Tijdens deze cursus ontdek je hoe je gegevens uit een database kunt uitlezen en tonen op een makkelijke manier. Je leert ook hoe je informatie kunt toevoegen, aanpassen of verwijderen zonder fouten te maken. Je gaat begrijpen hoe een DBMS werkt, wat tabellen en verzamelingen zijn, en hoe je alles logisch kan indelen. Beveiliging is ook een belangrijk deel: je leert hoe je data kan beschermen tegen mensen die er niet bij mogen. Daarnaast krijg je te zien hoe je een back-up maakt, zodat je gegevens veilig blijven als er iets misgaat. Na deze module kun je met SQL eenvoudige taken uitvoeren, data importeren en exporteren en begrijpen hoe alles samenwerkt in een applicatie.

Leeruitkomsten

  • Je kan gegevens uitlezen en tonen in een applicatie.
  • Je weet hoe je data toevoegt, aanpast en verwijdert.
  • Je begrijpt wat een DBMS is en hoe het werkt.
  • Je kan een database beveiligen tegen fouten of ongewenste toegang.
  • Je weet hoe je back-ups maakt en herstelt bij problemen.
