Module: Web Dev 1 - Fundamentals

Deze module gaat over de basis van het programering en het leren denken als een developer. Je leert hoe je simpele code schrijft, structuur aanbrengt in je project en begrijpt hoe verschillende programmeer talen toch veel met elkaar gemeen hebben.

Wat ga je leren?

In deze cursus leer je stap voor stap de fundamenten van programeren. Je ontdekt hoe verschillende programmeertalen werken en wat hun basisstructuren zijn. Je oefent met if/else, loops en funkties zodat je begrijpt hoe logica in code werkt.

Je krijgt inzicht in hoe talen met elkaar verbonden zijn. Hoewel elke taal zijn eigen regels heeft, zie je dat de concepten vaak hetzelfde blijven. Zo leer je sneller een nieuwe taal.

Een ander belangrijk deel is leren hoe je software goed structureert, door bestanden en mappen logisch te organiseren. Dat maakt je werk duidelijker en makkelijker te onderhouden.

Leeruitkomsten

  • Je beheerst de basisstructuren van minstens twee programmeertalen.
  • Je begrijpt hoe de talen met elkaar samenhangen.
  • Je kan software opdelen in meerdere bestanden en mappen.
  • Je kan eenvoudige code schrijven met variabelen, loops en funkt
